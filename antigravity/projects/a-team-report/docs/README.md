# A班集計システム (A-Team Report System)

## プロジェクト概要
このプロジェクトは、A班の作業データをリアルタイムで共有・集計するためのWebアプリケーションです。
Firebase Realtime Databaseを使用して、複数人による同時入力と即時集計を可能にしています。

## 主な機能
- **リアルタイム共有**: 各作業者の入力内容が即座に全ユーザーの画面に反映されます。
- **自動集計**: 入力されたRCAおよび混練のデータから、設備ごとの合計と発生量を自動計算します。
- **自動リセット**: 毎日17:00にデータが自動的にクリアされます（共同作業）。
- **モバイル最適化**: スマートフォンでの入力を考慮したUI設計。

## ディレクトリ構成
- `code/`: アプリケーションのソースコード (`index.html`)
- `docs/`: プロジェクトのドキュメント (このファイル)
- `../../../config/`: Firebase設定などの機密情報が含まれる設定ファイル

## セットアップと実行方法
1. `index.html` をブラウザで開くだけで実行可能です。
2. 同一ネットワーク内で共有するか、Webホスティングサービス（GitHub PagesやFirebase Hostingなど）にアップロードして使用します。

## 依存ライブラリ
- Firebase SDK (v9.22.0 Compat)
- Google Fonts (Inter, Roboto等)
